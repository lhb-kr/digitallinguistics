Quick Guide

This website helps you compare sentiment patterns in Shakespeare plays. Sentiment scores were computed from analysis text, but the website shows original Shakespeare text only.

1. Choose A View

  • Multiple: compare four plays at once with the same settings.
  • Overlay: place selected plays into one graph, then optionally add one comparison play.
  • Single: inspect one play in more detail with extra filters and charts.

2. Multiple Mode

  • Use each graph card's play selector to choose the four plays.
  • The shared settings panel applies to every multiple graph.
  • Use this view when you want to compare the overall emotional shape of several plays.

3. Single Mode

  • Sentiment Arc: shows sentiment across the selected play.
  • Scenes: shows average sentiment for each scene.
  • Act / Scene: heatmap overview by act and scene.
  • Speakers: average sentiment by speaker or character.

4. Overlay Mode

  • Select plays from the Selected Plays list.
  • Thin gray lines show the selected plays.
  • The thicker front line shows the equal-play selected average.
  • Use Comparison Play to add one separate play on top of the selected overlay.
  • The average uses normalized 0-100% play progress with 500 points, so long and short plays can be compared fairly.

5. Main Settings

Setting What It Does
Granularity Choose sentence, line/speech unit, or scene-level data.
Sentiment VADER is the default. Deep Learning is the second option. Both shows both scores together.
Smoothing LOWESS emphasizes larger trends. Moving Average uses a simple rolling average. Raw shows unsmoothed scores.
Window Controls how wide the smoothing neighborhood is. Larger values show broader trends.
Stage Directions Include or exclude stage directions from the graphs.
Y-axis Scale Auto zoom makes small changes easier to see. Fixed keeps the full -1 to +1 scale.

6. Reading Both Sentiment Scores

  • Line graphs use solid lines for both scores.
  • For scene, heatmap, and speaker charts, color means sentiment direction: green is positive and red is negative.
  • In Both mode, VADER is plain. Deep Learning is shown with slashes or hatching.
  • For paired bars, VADER appears first: left in scene charts, upper in speaker charts.

7. Single Filters

  • These filters apply only in Single mode.
  • Speaker, Act, and Scene filters narrow the single-play charts.
  • The Speaker chart can be sorted by most lines, most positive, or most negative.
  • Min count hides speakers with fewer than the selected number of speech units.

8. Downloads

  • Filtered CSV: exports the currently filtered single-play rows.
  • Play CSV: exports the public unit-level dataset for the selected play.
  • Scene CSV: exports scene summaries for the selected play.
  • All Summary CSV: exports play, act, and scene summary statistics for all 37 plays.
  • PNG: downloads the visible chart image.
